List
----

Basic Lists
-----------

Bullets
~~~~~~~

-  b1
-  b2
-  b3

Bullets
~~~~~~~

*  b1
*  b2
*  b3

List
~~~~

1. l1
2. l2
3. l3

Numbered
~~~~~~~~

#. a numbered list
#. and a second item
#. and a third item 

Nested Lists
------------

Bullets
~~~~~~~

* b1

  * c1
  * c2

* b2

  * d1
  * d2

* b3

Bullets
~~~~~~~

* this is
* a list

  * with a nested list
  * and some subitems

* and here the parent list continues

Malformed Lists that seem to work in HTML
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

* first item

* second item

    * a sub item that is spaced with four spaces rather than two
