Converting Notebooks to HTML
============================

+--------------------------------+-----------------------------------------------------------------------+-----------------------+
| Option                         | Description                                                           | Default               |
+================================+=======================================================================+=======================+
| jupyter_generate_html          | Enable sphinx to generate HTML version of notebooks                   | False                 |
+--------------------------------+-----------------------------------------------------------------------+-----------------------+
| jupyter_html_template          | Specify path to nbconvert html template file                          | ""                    |
